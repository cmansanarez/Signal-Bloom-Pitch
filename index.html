01

PROJECT OVERVIEW

Signal Bloom is an experimental audiovisual performance system that combines live coding, generative art, artificial intelligence, and improvisational performance into a single evolving environment.

Signal Bloom treats generation as a performative event.

  • INPUT_00AUDIO / LIVE
  • INPUT_01PERFORMER / LIVE CODING + IMPROV
  • INPUT_02AUDIENCE / ENERGY + INTERACTION
  • PIPE_00HYDRA → COMFYUI → ROUTER
  • OUT_00LIVE VISUAL OUTPUT / PROJECTION
02

THESIS

Signal Bloom explores how improvisation between human and machine actors creates conditions for emergence.

The project investigates the evolving relationship between:

  • PERFORMER
  • MUSICIAN
  • MACHINE
  • GENERATIVE SYSTEMS
  • AUDIENCE

The resulting experience is shaped through distributed authorship, where creative influence is shared across human, computational, and environmental participants.

03

THE QUESTION

AI-generated visuals are often treated as finished assets: generated, rendered, imported, and played back.

> audio.listen() // live > generate(prompt) // streaming > hydra.modulate(o0) // live > generation := performance // TRUE

What happens when generation becomes the performance?

ARTISTIC VISION

Signal Bloom exists at the intersection of performance, computation, and visual art. Its aesthetic language draws from signal processing, cybernetic systems, recursive structures, iridescent materials, and speculative forms of digital life.

The visual world feels synthetic, organic, unstable, and alive.

CRYSTALLIZATION ARC

Signal Bloom exposes the image generation process itself.

NOISE→STRUCTURE→FORM→IMAGE

Intermediate denoising frames are streamed directly into the visual environment. The audience witnesses emergence as it occurs.

The audience experiences the image through its transformation from noise to form.
